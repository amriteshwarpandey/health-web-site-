# health and fitness related -web-site- using frontend 
Myth Busting in Health and Muscle Building
This project is a simple, static website dedicated to debunking common myths related to health, fitness, and muscle building. It provides factual information to help users make informed decisions about their health and wellness journeys. The website is built using HTML, CSS, and a little bit of JavaScript.
